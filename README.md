# Copilot Marketplace

This repository is a starter marketplace structure for Copilot resources:

- skills
- agents
- hooks

No specific skills, agents, or hooks are included yet.  
Only the base folder layout and empty manifests are provided.

## Repository structure

```text
.
├── marketplace.manifest.json
├── skills/
│   └── manifest.json
├── agents/
│   └── manifest.json
└── hooks/
    └── manifest.json
```
